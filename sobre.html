Sobre este trabalho:

Este site foi criado pelo aluno Iago Luigi Boebel do curso de Análise e Desenvolvimente de softaware para a disciplina de Fundamentos da Programação Web Ele foi desenvolvido utilizando HTML, CSS e Javascript. As formatações de CSS foram feitas através de classes, os formularios são de preenchimento obrigatório através do atributo required do HTML e o tratamento do formulário foi feito através do javascrip lendo os parametros passados pela url com o método get. O site tem o intuito de no futuro ser o e-commerce da loja de bolos da minha mãe.
